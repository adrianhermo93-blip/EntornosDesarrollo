Este proyecto implementa una pequeña calculadora que clasifica la edad de una persona en distintas categorías: “Error”, “Adulto” y “Senior”.
La práctica incluye la creación y ejecución de pruebas unitarias con JUnit 5 y el uso de las herramientas de testing de Visual Studio Code.
Estructura del proyecto:
src/ main/java/entornos/CalculadoraRiesgo.java test/java/entornos/CalculadoraRiesgoTest.java pom.xml
El proyecto utiliza Java 17, JUnit 5 y Maven.
Ejecución de tests en VS Code:
1. 	Testing Explorer (icono del matraz)
Desde la barra lateral izquierda se pueden ejecutar todos los tests a la vez usando el botón “Run All Tests”.
2. 	Codelens Run  Debug
Encima de cada método con @Test aparecen las opciones Run y Debug, que permiten ejecutar pruebas individuales.
3. 	Depuración con breakpoint
Se puede colocar un punto rojo en el código de CalculadoraRiesgo.java y ejecutar un test en modo Debug para ver la ejecución paso a paso.

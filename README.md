Este proyecto consiste en una calculadora sencilla que clasifica la edad de una persona en distintas categorías: “Error”, “Adulto” y “Senior”.
La práctica tiene como objetivo aprender a trabajar con proyectos Java en Visual Studio Code, usar Maven, crear pruebas unitarias con JUnit 5 y manejar las herramientas de testing integradas en el editor.
Descripción del proyecto:
El programa principal se encuentra en la clase CalculadoraRiesgo, donde se implementa el método evaluarEdad.
Las pruebas unitarias están en la clase CalculadoraRiesgoTest, donde se comprueba el comportamiento del método con diferentes valores de entrada.
Tecnologías utilizadas:
• 	Java 17
• 	Maven
• 	JUnit 5
• 	Visual Studio Code
Estructura del proyecto:
src/main/java/entornos/CalculadoraRiesgo.java
src/test/java/entornos/CalculadoraRiesgoTest.java
pom.xml
Herramientas de testing utilizadas en VS Code:
1. 	Testing Explorer (icono del matraz)
Permite ejecutar todos los tests del proyecto desde la barra lateral izquierda.
Desde esta vista se puede ver el estado de cada prueba y su resultado.
2. 	Codelens Run  Debug
Encima de cada método anotado con @Test aparecen las opciones Run y Debug.
Run ejecuta solo ese test.
Debug permite depurar el test paso a paso.
3. 	Depuración con breakpoint
Se puede colocar un punto rojo en el código de CalculadoraRiesgo.java para detener la ejecución en una línea concreta.
Al ejecutar un test en modo Debug, el programa se detiene en el breakpoint y permite inspeccionar variables y el flujo del programa.
